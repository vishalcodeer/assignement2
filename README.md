GAME NAME-DODGE THE OBSTACLE

 Main Features

• Smooth left/right movement using arrow keys
• Random falling obstacles every round
• Lives system (3 chances)
• Fast-paced loop with live score display

 New Features Added

Highest Record Saving — your best score stays until you beat it

Replay System — after game over, press Y to instantly play again

Scoreboard Display — shows Score + Best Score together after every run

 How to Play (Simple Language)

Move your character (@) left or right using arrow keys.

Obstacles (O) fall from top to bottom.

Avoid them — if they hit you, you lose a life.

Every dodge gives you +10 points.

Lose all 3 lives → Game Over.

Beat your Best Score to improve your record.
